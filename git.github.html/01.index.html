Version Control System

Version control systems are a category of software tools that helps in recording changes made to files by keeping a track of modifications done in the code. Version conrol system are used many commands to keep changes in files.Git are the most popular version conrol system in the world.Git uses many commands like add,commit, clone,config,branch,cd etc.

Importance of Version Control System

As we know that a software product is developed in collaboration by a group of developers they might be located at differnt locations each one of them contributes to some specific kind of functionality/features.So in order to contribute to the product, they made modifications to the source code(either by adding or removing).A version control system is a software to make a product with teams and groups and make a communicate with each other related to the product or source code.Any person change the code and make a modification in the some projects with communicate with others developers.

  • Enhances the project development speed by providing efficient collaboration
  • Reduce possibilities of errors and conflicts meanwhile project development through traceability to every small change,
  • Employees or contributors of the project can contribute from anywhere irrespective of the different geographical locations through this VCS,
  • Helps in recovery in case of any disaster or contingent situation
  • For each different contributor to the project, a different working copy is maintained and not merged to the main file unless the working copy is validated. The most popular example is Git, Helix core, Microsoft TFS,
