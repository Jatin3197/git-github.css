Github---

Github is a cloud-based hosting service that helps the developers to store and manage or maintin their code,In the github to upload the files,folders and projects.In the github to host the projects from the local machine.It can also be tracked files and folders and make a changes in the code.In the github provides many features that make it easy for developers to merg of branches and make a collaboration or make a software projects in teams.After making a projects and files also have features to edit and rename the projects and files.

GitHub facilitates collaboration and version control in the following ways--

  • Branches and pull requests
  • Issues and tasks
  • Code reviews
  • Discussions
  • Version control
